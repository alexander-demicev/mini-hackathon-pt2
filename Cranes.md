# Cranes
Petya, Katya and Seryozha make paper from cranes. Together they made s cranes. How many cranes did each child, if it is known that Petya and Serezha made the same number of cranes, and Katya made two times more cranes than Petya and Serezha together?
The single line of the input file INPUT.TXT contains one positive integer S - the total number of cranes made (S <106).
In the single line of the output file OUTPUT.TXT, you need to output three numbers, separated by spaces - the number of cranes that each child made (Peter, Katya and Sergei).
